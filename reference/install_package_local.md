# Install package locally

Install package locally

## Usage

``` r
install_package_local(pkg_source_path)
```

## Arguments

- pkg_source_path:

  \- source path for install local

## Value

logical - package_installed

## Examples

``` r
if (FALSE) { # \dontrun{
results <- install_package_local("pkg_source_path")
print(results)
} # }
```
